Για να εκτελέσετε το εικονικό εργαστήριο, απαιτείται:

Υπολογιστής με Ubuntu 16.04 ή αντίστοιχο λειτουργικό σύστημα
Εγκατεστημένο VirtualBox
Εγκατεστημένο Docker
Επαρκής μνήμη και αποθηκευτικός χώρος


Εγκατάσταση του VirtualBox:


1)Κατεβάστε τον εγκαταστάτη του VirtualBox από την επίσημη ιστοσελίδα του VirtualBox: https://www.virtualbox.org/.
2)Ακολουθήστε τις οδηγίες εγκατάστασης για το λειτουργικό σας σύστημα Ubuntu 16.04.
3)Μετά την εγκατάσταση, εκκινήστε το VirtualBox και δημιουργήστε μια νέα εικονική μηχανή (VM) με τις απαιτούμενες προδιαγραφές (π.χ., RAM, αποθηκευτικός χώρος).



Εγκατάσταση του Docker:

1)Ανοίξτε ένα τερματικό στο Ubuntu 16.04.
2)Εκτελέστε τις ακόλουθες εντολές για να εγκαταστήσετε το Docker:
        sudo apt-get update
        sudo apt-get install docker.io
3)Εκτελέστε την εντολή docker --version για να επιβεβαιώσετε την επιτυχή εγκατάσταση.



Λήψη και Εκτέλεση του MySQL Replication Lab:

1)Κατεβάστε τον κώδικα ή τα Dockerfiles για το MySQL Replication Lab από το GitHub repository.
2)Μεταβείτε στον φάκελο όπου κατεβάσατε τον κώδικα.
3)Εκτελέστε την εντολή docker-compose up -d για να δημιουργήσετε και να εκκινήσετε τα containers του εργαστηρίου.
4)Περιμένετε να ολοκληρωθεί η διαδικασία εκκίνησης. Χρησιμοποιήστε την εντολή docker ps για να επιβεβαιώσετε ότι όλα τα containers τρέχουν κανονικά.
